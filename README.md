# English with Rangana - Website

This is a modern, responsive educational website template designed for "English with Rangana".

## Features
- **Responsive Design**: Looks great on mobile, tablet, and desktop.
- **Premium Aesthetics**: Uses a clean, professional color palette and high-quality typography.
- **Fast Loading**: Built with lightweight HTML, CSS, and Vanilla JavaScript.
- **Easy Customization**: All content is easily editable in the HTML file.

## File Structure
- `index.html`: The main structure and content of the website.
- `style.css`: All the styling rules, colors, and layout definitions.
- `script.js`: Handles interactivity like the mobile menu and scroll animations.

## How to Customize
1. **Text Content**: Open `index.html` in any text editor (like Notepad or VS Code) and change the text between the tags.
   - Example: Change `<h1>Speak English with <span class="highlight">Confidence</span></h1>` to your desired headline.
2. **Images**: Currently, the images are set in `style.css` using Unsplash URLs. To use your own images:
   - Place your image files in a folder (e.g., create an `images` folder).
   - In `style.css`, look for `.image-placeholder` and `.image-placeholder-vertical`.
   - Change `url('...')` to `url('images/your-photo.jpg')`.
3. **Contact Info**: Update the email, phone number, and links in the `Contact` section of `index.html`.

## Deployment
To publish this website, you can use services like Netlify, Vercel, or GitHub Pages. Simply upload these files to your chosen host.
